# Aperix Admin

Private internal dashboard for Aperix Studio.

## Purpose
- Track client websites and project ownership
- Monitor site health, deployments, and hosting
- Link repos, domains, credentials locations, and operational notes
- Serve as the internal command center for Aperix Studio

## Planned Areas
- Dashboard overview
- Client detail pages
- Repo and infrastructure view
- Lead intake and handoff workflow
- Health checks and alerts
